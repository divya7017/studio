# **App Name**: DataSight

## Core Features:

- User Authentication: Secure user login and registration using JWT.
- Filter Persistence: Store and reload user preferences using cookies.
- Date Range Filter: Interactive date range selection for data filtering.
- Age Filter: Dropdown filter to select age groups (<18, 18-40, >40).
- Gender Filter: Dropdown filter to select gender (Male, Female, Other).
- Feature Usage Bar Chart: Visualize feature usage with an interactive bar chart showing total click counts for each feature.
- Time Trend Line Chart: Display click trends over time for a selected feature, updated by clicks on the Feature Usage Bar Chart.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082), providing a sense of trust, intelligence, and authority, appropriate for analytics.
- Background color: Light Lavender (#E6E6FA), a soft, desaturated hue from the indigo family, offering a calm backdrop to emphasize data visualization.
- Accent color: Violet (#8F00FF), an analogous hue, more saturated and brighter, used for interactive elements and highlights to guide user attention.
- Headline font: 'Space Grotesk' sans-serif, for headlines and short amounts of body text.
- Body font: 'Inter' sans-serif, for longer amounts of body text. Use this font to provide clarity.
- Use minimalist, clear icons to represent different filter options and data points.
- Responsive layout to ensure the dashboard is accessible and functional across various devices.
- Subtle transitions when updating charts based on user interactions to provide feedback and enhance user experience.
